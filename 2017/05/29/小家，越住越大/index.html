小家，越住越大

  1. 家的不等式
    1. 房子不等于家，好好住的地方才是家
    2. 设计师不等于居住者，只能完成一半的设计，剩下的一半需要居住者根据自身生活习惯来完成；
    3. 更大的家不等于更好的家，关键是提高自己的住商；
    4. 家的面积不等于家的容积，懂得做好收纳的智慧；
  2. 给家减肥
    1. 贵买贱用才是最大的浪费；房子的空间价值是很贵的，不要被无用的物品所占据，从而影响空间的使用和空间价值的发挥；
    2. 克制购物欲，只买精品，将使用价值最大化； 不用的物品不轻易购买；
    3. 尽量让空间保持在三分之一的余量，这样才会让心理更好轻松感，而不会产生空间资源稀缺的焦虑；
  3. 收纳
    1. 四级收纳空间
      1. 房子
      2. 房间
      3. 家具
      4. 储物容器
    2. 做法
      1. 收纳规划：使用荧光笔
        1. 标注出可以用来做收纳的空间；
        2. 标注出各种柜子的位置和空间；
      2. 四条判断标准
        1. 各处均布，就近原则：任一空间都应分布有收纳空间；
        2. 占地面积12%以上；
        3. 立体集成：不宜零碎，尽量集成；
          1. 消除厚重感的办法
            1. 嵌入墙体；
            2. 浅色简洁；
            3. 隐藏接手；
        4. 二八法则：展示20%，收藏80%
          1. 展示美丽的，隐藏不美的；
          2. 展示常用的，隐藏不常用的；
  4. 容器
    1. 扔掉包装盒
    2. 使用统一的容器，原则：
      1. 易补货；
      2. 方形优先；
      3. 易于摞放
      4. 一目了然：透明
      5. 多处通用：避免专用形容器；
      6. 外表精致：例如藤艺对比塑料；
    3. 用容器管理物品，不适用容器的物品，避免购买；
    4. 给容器贴上标签；
  5. 玄关
    1. 鞋柜
      1. 标准层板深度350mm
      2. 上下层板高度基准间距160mm
      3. 打排钻，每隔320mm间距
      4. 构成：地柜，吊柜，高柜，鞋凳，装饰柜，前三者为基础配置，后两者为升级配置；
      5. 原则：
        1. 容量第一；
        2. 开敞常鞋位：底部两层为敞开式隔板；
      6. 鞋盒扔掉，改为防尘鞋袋替换，好处：轻巧不占地方，透明一目了然，方便出差使用；
      7. 出租房方案：买抽屉式鞋盒（带开窗），自做底架，堆叠摞放，容量很大；
      8. 客用拖鞋：采购4星级酒店使用的白色布拖鞋，可压缩叠放不占空间，可清洗
  6. 客厅
    1. 显大秘诀
      1. 活动家具不要大
      2. 固定收纳尽量大；
    2. 原则
      1. 眼不见为净，尽量隐藏，只展露少量的美物
      2. 统一规格的容器，例如：宽400mm*高350mm的柜格，使用4个收纳盒（5升/个），并贴上标签；
      3. 柜子底层开放式收纳格，放置长300mm宽350mm高300mm的布制衣箱，用来收纳玩具；
  7. 厨房
    1. 首选U型布局：动线更短，空间利用率更高；
    2. 避免让烟道打断黄金区；
    3. 冰箱大小跟厨房大小要匹配，小厨房用单开，大厨房再考虑双开，避免冰箱占据过大比例的面积；
    4. 橱柜
      1. 设备柜
        1. 消毒柜
        2. 水槽柜
        3. 灶台柜
      2. 储物柜
        1. 搁板柜
        2. 拉篮柜
        3. 抽屉柜
        4. 吊柜
          1. 收纳柜
          2. 油烟柜
        5. 高柜
    5. 台面：优先推荐石英石，其次是人造石（亚克力），严重不推荐瓷砖拼贴；
    6. 操作台高度：H1=身高/2+50mm
    7. 灶台高度：如果可能，考虑锅具本身的高度，比操作台高度低80mm左右最好；缺点是工艺复杂，费用升高；
    8. 不必花钱专辑
      1. 上翻柜门：打开后不容易够到柜门把手，导致利用率下降；
      2. 油烟机柜：如果被油烟机挡住不方便开，内部温度较高
      3. 微波炉柜：过高导致操作路线不方便，更推荐在视线以下位置的墙面做支架；
      4. 冰箱吊柜：冰箱会挡住路线，导致取物不方便，变成垃圾空间；
      5. 拉篮米箱：不易打扫，密封塑料米箱更加经济适用；
      6. 内置垃圾箱：容易生虫；
    9. 推荐款
      1. 抽屉和拉篮：争取配置2-3个抽屉；黄金区优先，其次可设置在高频次动作区域或狭窄分割的柜体区域；
      2. 消毒柜
      3. 洗碗机
      4. 盲柜/转角拉篮
      5. 吊柜底灯；
      6. 水槽周边：高抛龙头，净水器孔（省去台面打孔），沥水筐篮（比独立的沥水架更节省空间）
      7. 台面细节：墙面处使用圆弧角（易擦洗），挡水沿翻起5mm，预设滴水凹槽；
    10. 橱柜收纳
      1. 只收纳有价值之物，没有用的物品坚决丢弃；
      2. 重点清理对象：功能明显重复或需要换新的；热情消退打入冷宫的；早已没有存在感的；
      3. 上部：轻；中部：常用；下部：重
      4. 使用横杆和挂钩，让常用物品上墙，为黄金区留下操作面积
      5. 横杆安装：优先打孔，其次吊柜底装，再次使用伸缩杆；
      6. 吊柜间注意使用立体容器二次分装收纳：方形，透明
      7. 吊柜上层建议使用带把手的收纳箱可让取出更加方便，一次将整个箱子拉出；
      8. 地柜抽屉内部
        1. 使用分隔件对空间进行细分，提高使用效率；
        2. 竖放：查看和取出变得更加方便；
  8. 衣帽间
    1. 思考起床后的动线，通过设计和布置，让动线最短化；
    2. 塑料抽屉：
      1. 高度足够的抽屉，使得衣服可以立起来放进去，避免压出折痕，取出也更加方便；
      2. 利用隔件可以对抽屉内部再进一步的细分，提高空间使用效率；
    3. 柜门两侧安装首饰挂袋
    4. 脏衣篮放置需要干洗的衣服，一段时间后统一清洗；
    5. 鞋子：宜家白色布艺鞋柜，堆叠摞放；
    6. 镜前灯：旋转在镜子的上方
  9. 化妆台
    1. 玻璃两侧暗置竖向光源：左右打光均匀；
    2. 中间玻璃内置收纳：储存不常用的，缩短脸与镜子的距离；
    3. 两侧敞开，放置常用的；
    4. 底部增加两个抽屉，增加收纳空间；
  10. 家务
    1. 阳台上设置独立的家务区
      1. 杂物吊柜
      2. 洗涤用品开放式搁架
      3. 横杆挂衣架
      4. 挂钩挂置拖把和扫把；
      5. 洗衣机
      6. 手洗盆
      7. 地面抬高避免积水
      8. 加装铝合金百叶窗
      9. 多用龙头：分成三路出水
    2. 超细纤维巾
    3. 喷雾拖把
    4. 无线吸尘器；
