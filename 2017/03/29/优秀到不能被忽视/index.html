优秀到不能被忽视

1. 规则一：不要追随自己的激情

  1. 技能的精通是需要时间的，这才是最难的部分；各个行业都一样，重点在于，用正确的方法和最短的时间渡过这个时期，积累职场资本；
  2. 职业激情是稀缺的：只有4%不到的激情跟工作或教育相关，其余96%都只是某种爱好或兴趣，即不能创造可交换价值的活动；
  3. 激情需要时间
  4. 激情是精通的副产品；
  5. 获得动机需要满足的三种基本心理需求
    1. 自主：感觉对自己的生活拥有控制力，并感觉自己的所作所为是重要的；
    2. 胜任：感觉自己擅长于自己所做的事情
    3. 归属：感觉自己能够与他人建立联系

2. 规则二：工匠思维胜过激情思维

  1. 工匠思维：以产出为中心的职业观，关注自己给世界带来的价值
  2. 职场资本：个人所拥有的、在职场中属于稀缺而宝贵的技能；这是创建自己热爱的工作的关键通货；
  3. 成就大事的特质稀缺而宝贵，想获取这些特质，需要提供稀缺而宝贵的技能作为交换；
  4. 不适用工匠思维的三条特征
    1. 该工作无法让你发展稀缺而宝贵的技能从而与他人区别开；
    2. 该工作所关注的内容是无用的或者可能对世界有害；
    3. 该工作迫使自己与非常不喜欢的人在一起工作；
  5. 刻意的练习以跨越绩效高原：伟大的成就不在于天赋，而在于在正确的时间，正确的地点积累如此大的练习量；
  6. 刻意练习：通常由一位老师设计的，以有效改善某一个体某方面表现的唯一目的的活动；它要求将自身能力拓展到舒适范围以外，然后不断接收反馈；
  7. 假设只是努力工作，很快便会来到一个绩效高原，之后便无法取得任何进步；
  8. 投身于刻意练习之中，拓展能力范围，不断寻求反馈；
  9. 绩效高原：大多数一开始就活跃于专业领域的个体都会在有限的时间内改变自己的行为并且提升自己的绩效，直到达到某种可以接受的水平；在此之后，进一步的改善无法预知，而工作年数不足以预测一个人所能达成的绩效；
  10. 把时间花在重要的事情上面，而不是紧急的事情上面；
  11. 刻意练习的5大步骤：
    1. 判断自己身处哪一种职场资本市场
      1. 赢者通吃型：只有一种职场资本可以获取，并且很多不同的人在争夺这种资本；
      2. 拍卖型：有很多不同类型的职场资本，并且每个人可以生成他们自己独有的资本；
    2. 识别出自己要追寻的资本类型
    3. 定义优秀：需要有明确的目标，如果不知道目标，就很难采取措施；
    4. 拉伸与摧毁
      1. 拉伸：突破自己的舒适范围，进行严肃的刻意练习；
      2. 摧毁：积极接收真诚的反馈；
    5. 要有耐性：职场资本的获取需要时间；

3. 规则三：幸福来自于自主力

  1. 自主才是大家真正追求的东西；它是稀缺的，需要通过职场资本来换取；
  2. 自主力：对自己的工作内容和工作方式拥有发言权；
  3. 通过在工作内容和工作方式上赋予人们更多的自主力，会提高人们的幸福感、投入程度以及满足感
  4. 自主力陷阱一：自主力若不以职场资本而取得，则不具备可持续性；
  5. 自主力陷阱二：当你拥有足够的职场资本获取对职业生涯的合理控制时，雇主会设法阻碍你争取自主力；
  6. 信守财务可行性法则（要做有人愿意埋单的事情）：在决定是否追求某项有吸引力的活动时，应该问问自己别人是否愿意为之埋单；如果愿意，就继续追求；如果不愿意，就维持现状；

4. 规则四：使命感带来意义

  1. 使命：在创建自己热爱的工作时需要靠职场资本获取的另一个重要特质；它回答了“我的人生应该怎么度过”的问题，能够使你将时间都集中在有用的目标之上
  2. 相邻可能：在任何领域，下一个伟大的创意通常就出现在当前发展前沿之外的相邻区间，而这个区间包含了对现有想法的各种可能的新组合。关键是：你必须达到某一领域的前沿，然后，这种相邻可能以及它所包含的创新才会显现；
  3. 达到某一领域的前沿水平，需要在相当长的一段时间内专注于很少的几个课题。然而，一旦达到了前沿并且在相邻可能中到了一项使命，那么必须以极大的热情去追求它；
  4. 许多有着大量职场资本的人都可以在工作中找到很多不同的潜在使命，但却很少有人围绕这些使命建立自己的事业。要想运用好使命，仅仅达到前沿还不够，还需要善用“小赌”；
  5. 小赌：不一定要以一项伟大的创意开始，或者事先做好全盘规划，而是通过一系列有条不紊的“小赌”探索出一个可能不错的方向，并且从大量的小失误以及那些意义重大的小成功中汲取关键信息；
  6. 如果说职场资本能够让人找到一项有吸引力的使命，那么正是“小赌”策略让这项使命的成功实现成为可能；
  7. 要么引人注目，要么默默无闻：想要打造一份可以持续发展的事业，必须生产“紫牛”，即引人注目而且能够迫使人们广为传播的项目；
  8. 引人注目法则
    1. 迫使接触过它的人向他人进行评论；
    2. 它必须在一个利于这种评论发生的场合启动；

5. 正确地工作胜过正确的工作

不要执迷于寻找自己真正的“天职”，而要去掌握稀缺而宝贵的技能。一旦你积累到了这些技能所产生的职场资本，就要明智地运用好它。你可以用它来获取在工作内容和工作方式上的自主力，以及用来找到并实践某项改变人生的使命
