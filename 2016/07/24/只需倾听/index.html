只需倾听

  1. 基础知识
    1. 进化的三个阶段：爬虫动物、哺乳动物、灵长动物；
    2. 三个重要的器官：杏仁核、额叶、镜像神经元；
  2. 9条沟通法则
    1. 控制自己的情绪：包括害怕、愤怒、悲伤等
      1. 体察自己的情绪出现了变化；
      2. 在心里描述自己的情绪；
      3. 思考如何应对；
      4. 肯定自己的行动方案，鼓励自己；
      5. 开始行动；
    2. 清空成见
      1. 大脑出于本能和过往经验会形成初步判断，但这个判断不是结论，要时刻提醒自己需要下一步的交叉验证，而不是想当然或者先入为主；
    3. 让对方感受到我们的理解
      1. 体察情绪
      2. 告知体察并印证
      3. 修正体察（如有）
      4. 加深了解，包括程度和原因两个方面；
      5. 讨论行动方案
    4. 对别人感兴趣，而不是证明自己有趣
      1. 职场交谈的好问题
        1. 如何入行？
        2. 这一行最喜欢的地方是什么？
        3. 在职业生涯里，想要实现什么目标？
        4. 为什么这个目标对你很重要？
        5. 如果要实现这个目标，会促使你怎么做？
      2. 个人交往好问题
        1. 在你的生活中，对你影响最大的人是谁？
        2. 你最感激的人是这个人吗？如果不是，是谁呢？
        3. 有没有机会感谢那个人？
        4. 想象一下，你觉得人生尽善尽美是什么样子的？
      3. 陌生人的好问题
        1. FTD法则：感觉，想法，行为
        2. 问出让对方说出“我感觉…””我认为…“”我做过…“等三种回答的问题；
    5. 让别人感到自己有价值
      1. 寻求他人的认同感是人的天性；
      2. 让人们觉得自己有价值，与让其感到被理解和感到自己是个有趣的人不一样，因为这个做法在更深的层次上面触动了他们；
    6. 帮助他人抒发郁结
      1. 向讨论糟糕感受本身的方向去问问题；
      2. 让对方有充裕的时间表达；
      3. 不管对方说什么，不争论，不辩护；
      4. 等对方发泄完了，再简单多说一句：“再多跟我说说。”
      5. 其他可以问的问题：”我有没有让你觉得，我不够尊重你？，或者”我有没有让你觉得，你说的话不值得一听？
    7. 消除错位现象
      1. 定位问题，针对未来提供建议；
      2. 激情、热忱、自豪三个维度的调研与建议收集；
    8. 茫然无助的时候，主动示弱
      1. 示弱是一种强大的力量，比示强更能取得强大的效果；
      2. 每个人都有自己软弱的一面；
    9. 远离有毒的人
      1. 贪婪的人：转向离开，毫不犹豫；
      2. 恃强凌弱的人：对方以为我们好欺负，要表现出相反的特性，出乎其意料，并知难而退；
        1. 做法一：保持眼神接触，举止十分礼貌，但同时显示出感到无聊乏味的样子，放松，显示自己在走神一样；
        2. 做法二：进行反击，准备好退路；
      3. 爱占小便宜的人：以交换为条件；
      4. 自恋的人：可以选择离开，也可以选择留下，如果是后者，做好双方不平等的心理准备；
      5. 精神病患者：
        1. 特征：冷酷，缺乏同理心，以自我为中心，无情
        2. 做法：彻底离开
  3. 12个简单沟通秘诀
    1. 真的不可能吗
      1. 适用场景：当对方拒绝采纳建议进行改进的时候
      2. 做法：
        1. 借力：先让对方描述一件不可能做到，但做成了有极大好处的事情
        2. 卸力：表示理解
        3. 推力：询问怎么做可以把它变得可能
    2. 魔力悖论
      1. 适用场景：当对方拒绝沟通的时候
      2. 做法：
        1. 考虑如何描述情况，让对方说“是”；
        2. 当对方说是的时候，其潜意识里会觉得我们是让在他一边的；
    3. 催生同理心
      1. 适用场景：当A、B两方陷入沟通的僵局的时候
      2. 做法：把原来要问A方对B方的感觉的问题，让B方进行猜想并回答
      3. 原理：引导同理心发挥作用
    4. 出乎意料的立场转换
      1. 适用场景：当某人的表现让我们失望，需要与其交流以督促其改进的时候；
      2. 做法
        1. 预约专门的沟通时间（以便对方可以专注而不被其他事情打断，影响沟通效果）
        2. 讲3件自己可能给对方带来失望或者愤怒的事情，并表示歉意
        3. 询问对方自己讲得对不对
        4. 询问对方对这些事情的感受
        5. 结束对话，表示这些一直在心里，此次对话的目的只是为了把它们说出来，让双方变得更好，没有其他的事情；
    5. 你真的这么想吗
      1. 用来回应那些小题大做，夸张言辞的情况；
      2. 做法：
        1. 冷静、默数到5，问出问题，复述对方言辞（如果对方没反应过来的话）
        2. 再次确认对方是否真的这么觉得，如果是的话，希望听他好好描述以下情况，以便可以寻找解决方案；
        3. 分支
          1. 如果对方开始缓和，表示对其情绪（生气、郁闷、失望等）的理解，然后表达自己如何才可以帮得上忙；
          2. 如果对方坚定的回答“是”，那么坐下来详细聊一聊，因为可能真的碰到了棘手的大问题；
    6. “嗯”的力量
      1. 适用场景：将失去情绪控制的局面引导到理性思考的局面
      2. 做法
        1. 当对方发火的时候，不争辩，不反驳，而是引导对方多说说（借力，卸力）
        2. 在引导的过程中，对其情况表示理解，希望尽量帮其解决问题，期待多听对方讲讲发生的情况；
        3. 原理：通过这个引导，会让对方感觉自己跟他们是同一个战壕里面的，避免其处于反击状态，而会放下武器，同时也引导他们将情绪发泄出来；
    7. 事先自我揭短
      1. 适用场景：当人们对要发生的谈话存在疑虑的时候，缘于担忧我们的某个缺点或弱点
      2. 做法：
        1. 将他们担忧的问题说出来
        2. 讲一下应对的方案，或者一些可以证明其不成问题的证据
      3. 原理：事先摆出问题，就会让对方提前卸除对这个问题的怀疑和防御，然后专注于其他部分的内容；
    8. 从交易到交心
      1. 适用场景：开启深入内心深处的聊天，而不是例行公事
      2. 做法：问出正确的问题，这些问题包括
        1. 对人的期待、看法
        2. 对生活的期待、目标、看法
        3. 对工作的期待、目标、看法；
        4. 对未来的展望、看法
        5. 对事物的想法
    9. 肩并肩的交流
      1. 适用场景：通过非说教的方式引导对方对问题进行思考， 以得出答案；或者为了得到更多的信息；
      2. 做法：跟对方一起做一件事情，然后用预先想好的问题进行提问，探索对方在做什么，想什么，有什么感受；
      3. 原则：不争论、不说教、不打断；
    10. 填空法
      1. 适用场景：打破对方的防御状态，让对方放下过戒备，参与到对话里面来
      2. 做法：询问问题，但将关键部分放空，邀请对方回答
      3. 原理：没有推销什么，而是引导对方自己说出来；
  4. 快速应对7种棘手问题
    1. 如何搞定糟糕的团队
      1. 先团队沟通
        1. 肯定大家的价值，放低自己；
        2. 提出问题
        3. 提出解决方案
        4. 讨论愿景
        5. 讨论共同价值观
      2. 再逐一单独沟通
        1. 骨干：为其扫除障碍；
        2. 老员工：让其感觉到自己有价值；
        3. 抱怨者：让其感觉到自己很重要；
        4. 同行者：挑明对方的想法，用宽容和谦卑消解；（重量级感谢）
    2. 如何顺利升迁
      1. 问两个问题：
        1. 你最希望我持续去做的三件事是什么；
        2. 你最希望我永远不做的三件事是什么；
      2. 问一些能够交心的、加深双方感情的问题；同理心，让对方感觉到你理解他；
      3. 把眼光越过上级，往外寻找更成功更受尊敬的人士来做导师，询问：我想学习你懂的一切，最好的方法是什么？
    3. 如何对付自恋狂
      1. 将任务的优先级安排交给对方来决策，并用较正式的方式记录下来；
      2. 避免陷入由其提要求，却不用为决定负责的境地；
    4. 如何拓展人脉
      1. VCP流程：相识、可信、互惠；
      2. 相识阶段
        1. 对别人感兴趣，而不要证明自己有趣，多谈别人，少谈自己，多问别人是怎么做的，如何做到，什么样的策略有效等；
        2. 让人们感觉到你的理解：表露你的关心，理解别人的处境，帮助解决；
        3. 问出交心式的问题
        4. 重量级感谢
      3. 可信阶段
        1. 不要制造错位，不要做出无法实现的承诺，坦诚而无误的呈现自己；
        2. 让对方觉得自己有价值，尽力去帮助对方；
        3. 如果得到帮助，要明确表示感激；
      4. 互惠阶段
        1. 让新朋友一直觉得他很有趣，觉得他很重要，觉得你能理解他；
        2. 放松，让时间来自然培养和加深双方的关系；
    5. 如何对付失控的人
      1. 第一阶段：从爬虫到哺乳
        1. 告诉我出了什么事
        2. 复述对方的意思
        3. 等待，直到对方说出“是”
        4. 用一个准确的词，描述对方的情绪感受，再得到“是”；
      2. 第二阶段
        1. 马上解决这个重要是如此重要，原因是…（让对方填空）
        2. 指出道路，表明要一起去寻找解决办法；
    6. 如何与自己对话
      1. 六步暂停法
        1. 练习对身体的觉知：辨认身体的感觉（五官）
        2. 练习对情绪的觉知：给情绪命名
        3. 练习对冲动的觉知：“这种情绪让我想…”，觉察自己冲动
        4. 练习对后果的觉知：要是我一时冲动做了，会发生什么事？
        5. 练习对解决办法的觉知：更好的做法是…
        6. 练习对好处的觉知：如果我采取了这个更好的做法，好处是…
    7. 如何接近大人物
      1. 创造一对一的交谈机会
        1. 让对方感受到你的理解
        2. 让对方感觉有面子，创造镜像神经元的同理心，鼓励他们产生回报你的愿望；
      2. 利用网络
        1. 在线上与大人物接触
        2. 核心原则：人们希望得到别人的理解
        3. 做法：在他会去关注的地方，表达自己的理解
      3. 接近守门人
        1. 表达对守门人的职责和工作的重要性
        2. 发现守门人的有趣之处
        3. 让守门人感受对其工作的理解
      4. 三个基本原则
        1. 让人们觉得自己很有趣
        2. 让他们觉得自己很重要
        3. 让他们感受到你的理解
