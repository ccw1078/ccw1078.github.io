美国国会

  1. 大妥协
    1. 议会缘何不可？
      1. 没有采取一院制的原因：为了避免过分的民主；因为财产持有者们担心不受约束的多数暴政可能给自己的利益带来损害；
      2. 未采纳议会民主制
        1. 优点：保证行政分支的独立性，避免其受到国会多数党变化的影响；
        2. 缺点：决策过程更加拖沓、烦冗、低效
          1. 但这正是制宪者们的初衷，为了避免草率行事，调和舆论突变、防止权力过分集中；
    2. 一切必要且恰当的法律
      1. 一切立法权
      2. 钱袋权：批准联邦支出，以及征税；
      3. 贸易条款：调节各州以及其他国家之间的贸易往来；
      4. 外交和军事政策方面则不太明确
        1. 召集三军；
        2. 提供军费；
        3. 宣战；
    3. 人民的众议院
      1. 435名议员，还有6名特别代表（自治邦、属地、特区）
        1. 这些代表不能在议场上投票，但可以在委员会里面投票；
      2. 实名制投票；
      3. 为应对日益复杂的经济和社会问题，产生了委员会制度；议员也由兼职变成了全职；
      4. 2年选举一次（全世界最短的任期）
    4. 冷静的参议院
      1. 相对众议员，由于参议院投票规则的不同，使得每个参议员拥有更大的影响力；
      2. 众议院的规则对组织化的多数党有利；参议院的规则，则赋予了少数派更多的力量；
      3. 任期6年，第2年换选3分1；
        1. 由于任期较长，参议员可以更长远的考虑和决策，而不用担心受到选民的影响；
        2. 可以利用时间来证明自己的正确性；同时随着时间推移，很多选民也会遗忘当初的议题；
      4. 早期参议员会州选举，被诸多利益群体影响，1913年第17条宪法修正案将参议员改由民众直选；
      5. 由于每2年只换选3分之1，意味着每次换选，原来的阵容仍保持不变；这使参议院的各项规则稳定不变，不会像众议院每次重选后，新晋多数党会调整规则以有利于自己；
      6. 反对者一般使用阻挠议事的方法，使得法案不能被简单多数通过，而是需要达到一定比例票数，才能终止辩论（早期为三分之二67票，后来改为五分之三60票）
    5. 在让步中前行
      1. 立法过程是一场旷日持久的协商谈判；只有协同工作，并懂得让步，才有可能推进任何一件事情；追求完美拒不让步，反倒一事无成；
  2. 竞选与选民
    1. 竞选国会议员
      1. 代表？还是带领？议员需要就重大问题的投票做出自己的选择，并在回到家乡后，向选民解释他们的选择；
      2. 议员需要花费很多时间在筹集竞选资金上面；因此在国会的工作时间压缩到了周二到周四；
      3. 限制竞选资金的两难
        1. 正方：富人有钱说话更大声；
        2. 反方：限制了言论自由；
      4. 众议员竞选并不激烈，部分原因在于在任者控制了选区的划定，使得大部分地区趋于稳定；
      5. 每次选举后，众议员队伍的调整率约10-20%；半数众议员任职时间少于11年；
    2. 国会中的政党
      1. 政策各自选择本党的成员，参加到各种委员会当中；
      2. 多数党筹划并推动立法进程；少数党则抓住每个时机重塑议程；
      3. 由于选举人团和胜者通吃制度的存在，使得想要赢得选举的政党都必须具备广泛的民意基础并具有包容性，同时使得专注于意识形态的第三党和地区性政党难以发展壮大；
      4. 为了获得支持，议员本身也需要加入某一政党进行抱团，才有可能展开和推进计划；孤身奋战则难有成就；
      5. 政党做为组织，具有让自己长期生存下去的生命力，本身也会积极去影响各种结果，体现自己存在的价值；
    3. 新议员群体
      1. 国家利益、政党团结以及总统施加的压力，经常迫使议员面临投票的艰难选择；
      2. 少数族裔的两难
        1. 分散居住，由于比例少，无法选出自己的代表；
        2. 集中居住，则会形成种族隔离；
    4. 服务选民
      1. 议员基本职责：立法、传达和提出主张
      2. 立法：举行听证会，并提交法案以进行投票表决；
      3. 传达：向选民通报议题，并解释采取的立场；
      4. 主张：在国家事务上为选民的利益和看法做出辩护
      5. 选民服务
        1. 答复选民的请愿和问题；
        2. 保荐：推荐某个人员到行政分支、军事院校、法院等机构任职；
    5. 媒体关系和新闻报道
      1. 竞选能否获胜需要依赖媒体；
      2. 众议院从第一天开始即对媒体开放；从1979年开始，允许使用电视播放辩论过程；
      3. 记者招待会是国会大厦日常工作的一部分；
  3. 委员会其内
    1. 变法案为法律
      1. 法律法规主要是在委员会内制定出来的，所以委员会是很重要的，是整个立法程序的核心；新议员都会申请被任命到某个委员会中，以便发挥更大的作用；
      2. 由政党指导委员会任命议员到委员会中；每个政党可以获得的会员会席位数与政党议员席位数比例大致相当；
      3. 委员会来自不同层面的运作资金用以雇用工作人员；
      4. 委员会内部又分为小组委员会，以便进一步实现专业化，并由它们承担绝大多数工作；
      5. 众议员提交法案，书记员进行编号，之后由法规专家根据规则将法案移交到适当的委员会进行处理；
        1. 同一个法案，在众议院可能提交给多个委员会；但在参议院只会提交给某一个委员会；
      6. 立法者最大的成就是阻止糟糕的想法获得通过；只有3%左右的法案最终变成法律；
      7. 国会生涯的标志性成就是以名称为人熟知的法案；
      8. 在举行听证会过程中，工作人员选择的证人很重要；如果选择得当，问题合适且妥善作答，将决定辩论形式及其产生的法律；
      9. 多数法案获得通过时，与委员会提交的版本相关无几；这也间接说明了委员会的发言权巨大；
        1. 法案在委员会一般要经过严格审核，并已经通过协商达成某些共识，所以才不太需要修改；
    2. 众议院委员会
      1. 部分重要的委员会（如筹款、拨款、规则等）要求其成员必须专职，不得兼职其他委员会；
      2. 部分委员会的职责存在重叠，导致一个法案可能递交给多个委员会，同时不同委员会也可能给出矛盾的意见；
    3. 参议院委员会
      1. 早期除了几个重要的委员会外，多数委员会实在挂名，其间没有什么利益可图，挂名的目的是为了获得一个办公室；
      2. 参议员同众议员一样，因共同利益被吸引到委员会中；
      3. 参议员比众议员服务于更多的委员会，更像通才而非专才；
      4. 部分委员会的成员倾向于专注立法工作而不寻求全国关注，把干扰降到最低；
      5. 某些委员会则特别喜欢吸引公众注意力，例如司法委员会；
      6. 由于大法官通过法律解释越来越多的参与到政策制定的过程中，使得参议院与总统就大法官提名之间出现了较量；
    4. 拨款：万事之源
      1. 拨款法案在每年10月1日前交予总结签署；但国会很少准时，因此只好通过继续生效决议，延续前一年的支出安排，以便争取额外的时间；
      2. 不可预见的支出，例如救灾款，则一般通过补充拨款进行处理；
      3. 占大头的支出包括：社会保障、医疗保险、退伍军人年金、国防开支等；
      4. 《预算法》中有一项调节条款，它对法案的辩论时长作出了限制；其本意是为了调节预算委员会和拨款委员会宰的分歧，后来由于它能够避开阻挠议事行动，使其成为一项立法策略被加以利用；
      5. 两院的规则均限制在拨款方案的基础上进行立法；
      6. 拨款法案由众议院制定，参议院审批，因此众议院实际上占据更大的主动权；
      7. 预算决定着两院拨款委员会能够分配的资金额度，它们将总额在下属的十几个小组委员会间进行划分；各小组委员会再对其全部份额加以划拨；小组委员会同时负责监督联邦支出的一些具体领域，如农业、国防、能源等，其威望和权力很大；
      8. 总统和国会在拨款问题是博弈的关系，有如斗鸡；
      9. 拨款法案中，有一种叫做专项拨款的东西，可以用来向某机构提供一笔动作资金，因此它也成为游说人员的目标，将其视为满足主顾需求的最直接方式；
  4. 议场之上
    1. 辩论、规则与程序
      1. 众议员更多在委员会内部发挥影响力，跨委员会则影响力很弱；参议员则可以在任何方案提出意见；
      2. 党纪长期以来一直在国会中难以贯彻，因此重要法案经常需要构建跨党联盟以争取能够顺利通过；
      3. 议员需要在议场上就提案进行发言，但实际上听从很少，仅有的听从还经常是排队等待发言的同僚；议场策略更多取决于投票表决，而不是能言善辩，因此实际上议场上偶尔才会出现辩论；辩论的目的其实更多在于通过电视转播，赢取公众的注意力，从而为少数派争取到更多的支持；
      4. 相比三权中的另外两个分支，国会的活动更多的出现在公众的视野中；每日讨论的所有事情，都会记录下来，并在第二天的《国会记录》中进行印发；
      5. 众议院的议场规则很多，多到议员都难以搞清楚，从而需要求助于议场专家的工作人员；参议院的规则很少，但在实践中却很少遵守；
    2. 众议院多数党的支配地位
      1. 工作流程
        1. 各委员会上报待讨论和投票的法案清单到规则委员会
        2. 规则委员会起草决议（称为特别规则），确定提交到议场的修正案数量、类型，并分配辩论时间；
        3. 众议院对特别规则进行投票；
      2. 由于众议院规模较大，因此众议院不允许采用阻碍议事进程和其他拖延策略，这使得多数党有机会中断讨论并进行投票，因此，多数党在众议院处于支配地位；
      3. 但法案优势不明显，陷入僵尸时，可以通过提出弱化部分条款的修正案，以让其得以投票通过；
      4. 当为了避免方案通过，可以通过提出包含过激条款的修正案，从而使得支持者知难而退从而放弃；
        1. 这种策略有时候会适得其反，例如1964年增加妇女平等的修正案，最后却通过了；
      5. 法案的反对者可以使用引入与法案主题的修正案，以挑起冲突；这种冲突的目的在于迫使某些摇摆选区的议员对这些议题发表意见，而这种意见可能会影响其连任，从而制造了压力；
      6. 有些投票表决具有所属政党与地方利益的冲突性；此时众议员通常倾向地方利益；政党领袖也难以要求议员自断前途；
      7. 众议院的议程据说很杂乱；原因估计于议员人数众多，代表不同选区，多种利益互相纠缠；
    3. 参议院少数派的威力
      1. 参议院的动作与众议院很不同；整体来说要有序很多；
      2. 规则委员会在众议院的权力很大，决定了重要法案的命运；但在参议院，这个委员会主要负责分配办公室和停车位；
      3. 参议院的大部分事务都通过“一致同意”的方式进行处理；这种方式可以节省很多时间（这也是为什么参议院的规则不被遵行的原因）；同时这也使得单个参议员能够发挥很大的影响力，因为只要其站起来提出反对，“一致同意”则失效，接下来就需要按规则行事，从而拖延了时间；
      4. 一般来说，若某个参议员反对某个法案，只需要私下告知政党领袖即可，这种方式叫“推迟”策略，这样多数党领袖就不会宣读该法案，以免浪费时间；待“推迟”得到解决后，再来宣读；
      5. 由于参议院的表决票数通常很接近，这使得游说机构非常重视搞定参议院；
      6. 由于议员每周在国会只工作3天，余下日子回老家，因此议员若采用阻挠议事，则会造成很大的时间成本威胁；当采用阻挠议事时，马上终止辩论，进入投票；当达不到60张赞成票时，法案将视为无效不得通过；为避免出现这种情况，法案需要有跨党派的妥协、退让与合作，才能得以顺利通过；
    4. 立法如灌肠
      1. 一部法案的出台，背后是各种利益的博弈和交换；例如议员之间也会存在投票交换；
      2. 参议两院的投票时间约为15-20分钟；在投票结束前，议员可以改变投票；领导层一般会保留一些投票，然后观察投票结果，若胜利有望，则放开这些保留票，让其随便投；
      3. 即使法案在两院得到通过，也不代表程序结束，因为两院的法案若存在差异，双方需要进行协商，其中一院必须接受另一院的版本；
        1. 当一院对某项法案进行表结后，利益未得到满足的各方势力，会求诸另一院，以争取一些条款的改变；
      4. 在两院协商过程中，众议员更有优势一些，因为其专职少数委员会，因此更加精通某个领域的知识；
      5. 如果一个政党同时控制两院，则联合起来的领导层可以绕过协商委员会，直接在两院间达成解决方案；
      6. 一旦协商会议就妥协做出报告，两院均必须直通通过协商会议的版本，且不再予以修正；
      7. 两院通过的法案，总统有否决权；参议院需要有三分之二的票数，才能推翻总统的否决；因此法案的制定过程中，一般也会考虑总统否决的可能性；
  5. 制约与平衡
    1. 主要立法者
      1. 总统是主要的立法者，这点在每年初的国情咨文中表现得最明显；一般在众议院的议场举行，参议员会到众议院的议场一起参加；
      2. 总统作为主要立法者的角色，始于伍德罗威尔逊，此前的总统更多定位为已通过的法案的执行者和管理者；
    2. 意见与同意
      1. 总统做出的高层任命需参议院同意；
      2. 参议院礼遇惯例：参议员有权力反对其所属州的被提名人，使之不得被任命；
        1. 现在改成需要该州的两名参议员同时反对才有效；
      3. 为避免提名遭到参议院否决的尴尬，总统在提交名单前，会提前找参议院协商意见；因此，很多时候被提名人并非最佳人选，而是双方都没有意见的人选；
      4. 总统有权在国会休会期间做出任命，以避免行政停顿；这项权力现在被总统利用来做一些有争议的人员任命安排，只等国会一休会，总统马上宣布一批休会任命；
      5. 由于行政官员的任期不会超过总统，所以国会的争议较少；但最高法院的司法提名则争议很多，因为大法官的任职是终身的；
    3. 战争与和平
      1. 国际事务因为与地方利益的关系较少，国会因此对总统的干预比较少
      2. 国会控制着钱袋，理论上可以通过停止拨款终止战争，但这也会让他们陷入弃战场上的士兵于不顾的道德风险；
      3. 在国家安全事务方面，由于面临公众的压力和风险，国会比较难对总统的决策进行干预和阻拦；
    4. 国会调查
      1. 国会有权发起对总统在外交、军事或国内政策上不法行为的调查；从而对行政分支起到了监督和制约的作用；
      2. 调查也会影响议员的政治生涯，使其收获公众的注意力和选票支持；
    5. 惩罚与保护
      1. 对政府不法行为的惩罚为弹劾官员，包括总统在内，同时也包括司法分支的联邦法官；
      2. 议员享有“发言和辩论”条款的保护，即政府不得以其辩论时的言论起诉国会议员，也不得在议员前去参加国会的途中将其逮捕；
        1. 此条款的目的是避免行政分支干预立法活动；
    6. 国会与法院
      1. 司法系统在国会与行政分支中间扮演仲裁者的角色，其有权裁定国会法案或者总统的行为违宪或者违法；
      2. 当法院做出违宪裁定时，理论上国会可以通过制定宪法修正案来反对裁定，但由于宪法修正案需要两院通过，还需要各州批准，因此事实上如果事情不是广泛达成共识，修宪其实是非常困难的；过去200多年，提出的上万条修宪法案中，只有27项最后通过；
      3. 法院允许国会将部分权力委托给独立监管委员会，但不可委托给非政府的商业团体，也不可委托给行政分支；
  6. 国会建筑群
    1. 国会大厦
    2. 国会议员
      1. 议员在国会山会被区别对待，有专用的设施，如休息室、停车场、餐厅等；
      2. 议员出身的职业很多样，总体来说律师和企业家占更大的比例；
      3. 议员的两难问题
        1. 他们由地方选举出来，若过分追求地方利益，会被批评地方保护主义；
        2. 若站在地方民众的对立面，却需要冒着失去议席的风险；
    3. 工作人员
      1. 新任议员的首要工作是雇用工作人员并设立办公室；
      2. 议员的工作班子大都是具有政治头脑的年轻人；工作人员的报酬很少，但积累的工作经验却非常重要，可以成为日后的跳板；
    4. 游说人员和其他访问者
      1. 游说团体的办公室主要集中在 K 街，因此也称为 Ｋ街游说人员；
      2. 游说人员背后代表着各种特殊利益团体，但他们却是立法程序的合法组成部分，因为这种利益团队也代表着选票的渠道，他们提供了信息的来源和利益的诉求，同时也很有可能提供竞选资金的募集；
        1. 立法铁三角：委员会主席、行政高级官员、选民；
      3. 记者与议员是相爱相杀的关系，记者需要议员提供消息来源，同时保持自身的独立性；议员需要记者帮忙做宣传；
      4. 研究人员也是重要的组成部分；少数学者甚至参与了选举并获得了成功；
      5. 美国公民一生平均去两次国会，一次在孩提时代同家与或同学一起；一次则是作为父母带家人前来；
